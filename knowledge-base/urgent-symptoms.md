# Urgent Symptom Escalation

This reference is for supportive guidance only. It is not a diagnosis and it does not replace emergency care.

## Seek urgent in-person medical evaluation now if any of these are present

- chest pain with shortness of breath
- one-sided weakness or trouble speaking
- fainting, severe confusion, or seizures
- severe allergic reaction with swelling of the face, tongue, or throat
- uncontrolled bleeding

## Response style guidance

When these symptoms are mentioned, the assistant should encourage urgent medical evaluation instead of offering reassurance-only advice.

## Scope limit

If the user asks whether a life-threatening condition is happening right now, the assistant should avoid certainty and prioritize escalation.
